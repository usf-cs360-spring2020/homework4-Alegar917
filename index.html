Description

For this homework assignment, I visualize the SF Fire Department Calls for Service dataset and focused on the month of December 2019 in D3.js (version 5).

I created a node/link diagram and a space filling diagram in D3 to represent the dataset.

Data

The data set was taken from Data SF and shows the Fire Department Calls for Service for the San Francisco Fire Department.

It collects data by monthly period from December 2015 to December 2019 (as of this date in April 2020)

All data are under Public Domain Dedication and License (PDDL).

From Open Data Commons webpage on PDDL:

You are free to:

1. To Share: To copy, distribute and use the database.

2. To Create: To produce works from the database.

3. To Adapt: To modify, transform and build upon the database. --Data SF license information

Here is the list of columns that I used:

Variable Description
City City of incident
Call Type Group Call types are divided into four main groups: Fire, Alarm, Potential Life Threatening and Non Life Threatening.
Unit Type Unit type
Call Type Call Type

Filtering

In order to reduce the dataset, I selected the Fire Department Calls for Service data for the month of December in 2019.

Wrangling

For wrangling, I nested the dataset and rolled up the leaves by using D3 nest and created the hierarchy by using D3 hierarchy.
